# DevStackBox

The modern local development environment for Windows.

DevStackBox is a free and open-source desktop application that bundles Apache, PHP, MySQL, and phpMyAdmin into a single native Windows application. It is designed to provide a clean, reliable, and developer-friendly local development experience without unnecessary complexity.

## Projects

- 🚀 **DevStackBox** - Desktop application for Windows
- 🌐 **Website** - https://www.devstackbox.com
- 📚 **Documentation** - https://www.devstackbox.com/docs
- 📦 **Latest Release** - https://github.com/DevStackBox/DevStackBox/releases/latest

## Features

- Native Windows desktop application
- Apache, PHP, MySQL, and phpMyAdmin included
- Multiple PHP versions
- SSL and Virtual Hosts
- Configuration editor
- Automatic updates
- No telemetry
- Offline-first
- MIT Licensed
